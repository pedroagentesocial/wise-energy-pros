# Public assets

Esta carpeta es para archivos estáticos.

Estructura recomendada:

- `/public/images/` → imágenes (`.jpg`, `.png`, `.webp`, `.svg`)
- `/public/videos/` → videos (`.mp4`, `.webm`)
- `/public/icons/` → iconos (`.svg`, `.png`, `.ico`)

Uso en el proyecto:

- Ruta en código: `/images/mi-imagen.webp`
- Ruta en código: `/videos/hero.mp4`
- Ruta en código: `/icons/logo.svg`

En GitHub Pages también funcionará porque Astro aplica el `base` automáticamente.
